# Phase 0: CPO Briefing - Productivity Mind-map Task Manager

## 1. Product Briefing & CPO Assessment
**Project:** Visual Mind-map to Kanban bridge.
**Core Mission:** Closing the "Ideation-to-Action" gap.

### The "Creative-First" Assessment:
Users brainstorm in Mind-maps (Whimsical/Miro) but they execute in Lists (Linear/Asana). The "Export" process is where projects die. We are building an **"Infinite Canvas Task-Linker."** Every bubble in your mind-map is a live task; every line is a real dependency.

### Key Hooks (LTV/CAC Efficiency):
- **Dynamic Task Conversion:** Click a node to "Action it"—it instantly gains a due date, owner, and status while staying on the map.
- **Dependency Flow Visualization:** See exactly which "Brainstorm Bubble" is blocking the "Launch Bubble."
- **AI-Subtask expansion:** Right-click a "Big Idea" bubble and the AI generates the first 5 actionable sub-tasks as child nodes.

## 2. Raw Data Audit Trail (Research)
- **Reddit/Hacker News Insights:** "I have great ideas in Miro but I never open the Trello board to actually do them." "Linear is too rigid for the messy start of a project."
- **Competitors:** Whimsical (Great visual, weak task logic), Miro (Too heavy for simple tasks), ClickUp (Mind-maps are an afterthought).
- **Market Gap:** A tool that treats the *Canvas* as the primary workflow engine, not just a drawing board.

## 3. High-Level Requirements
- **Swarm Activation:**
    - **CTO:** Build a high-performance vector graphics canvas that can handle thousands of "Live Task" objects.
    - **CMO:** Target "Product Designers" and "Founders"—the "Messy Middle" of project starts.
    - **COO:** Freemium model—Free for personal ideation, $12+/user for Collaborative Execution.
