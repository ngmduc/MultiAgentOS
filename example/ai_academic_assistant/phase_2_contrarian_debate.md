# Phase 2: Contrarian Debate - AI Academic Writing Assistant

## 🥊 Round 1: Cross-Examination (Bắn Tỉa Lẫn Nhau)

**CTO attacks CMO:**
"CMO wants to market this as 'Anti-Cheating AI' for Lab Leads. This is a PR disaster. If we call it that, researchers will be insulted, thinking we imply they *are* cheating. We should market 'Integrity-as-a-Service.' How do we avoid the negative 'policing' connotation?"

**COO attacks CTO:**
"CTO proposes an 'OCR/Parsing Engine' like Grobid. These are notoriously slow and memory-intensive. If we run these on every user upload, our server costs will balloon, making the $49/mo plan unprofitable. Can we use a cloud-native PDF parser (AWS Textract) instead?"

**CMO attacks COO:**
"COO suggests a $250 'Per-Grant' fee. Grant writing is a stressful, low-certainty period. Asking for a large upfront fee when their budget is tight is a barrier to entry. Why not a $15/mo 'Researcher Tier' and a $499 'Institutional Suite'?"

**CPO Direction:**
"CTO, Use Textract for speed but Grobid for structural metadata (citations). COO, Institutional sales are where the money is; start with the 'Researcher Tier' as a loss leader. CMO, focus on 'Audit Readiness'—the tool is to help them survive a peer review, not a police check."

---

## 🛡️ Round 2: Defense & Pivot (Bảo Vệ & Cải Tiến)

**CTO Defense:**
"I will pivot to a **'Hybrid Ingestion' model**. AWS Textract for fast text extraction, and a background 'Citation-Fixer' using Grobid only for the identified reference blocks. This keeps the UX fast and the server costs low while maintaining 100% accuracy on citations."

**CMO Defense:**
"Pivoting the brand to **'ScholarShield'**. The message is: 'Protect your legacy from accidental errors.' We aren't checking if you cheated; we are checking if you were *careful*. The 'Audit Tool' is now the 'Submission Readiness Check'."

**COO Defense:**
"Accepting the **'Freemium-to-Insto' funnel**. $19/mo for individuals to build the dataset, then we sell the 'Departmental License' ($5,000/yr) to Deans and Lab Leads who want to ensure their entire team's outputs are 'Shield-Verified'."

---

## ✨ Round 3: Final Polish (Mài Giũa Cuối Cùng)

**Final Alignment:**
- **Tech:** Verification-first RAG with a focus on 'Citation Anchor' integrity + Hybrid PDF parsing.
- **Marketing:** 'ScholarShield'—Submission-ready papers with zero hallucination risk.
- **Commercials:** $19/mo Individual -> $5k/yr Institutional.

**CPO Conclusion:**
"A solid, defensible niche in the AI space. We solve the 'Trust Deficit' in academia. Proceed to Phase 3 Master Blueprint."
